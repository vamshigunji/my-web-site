# Venkata Gunji | Data Scientist & Developer Portfolio

A modern, high-performance personal portfolio built with Next.js 15, Tailwind CSS v4, and Framer Motion.

## Features

- **Next.js 15 (App Router)**: Cutting-edge framework with React Server Components.
- **Tailwind CSS v4**: CSS-first configuration using the new engine.
- **Framer Motion**: Scroll-linked animations and smooth transitions.
- **Dark Mode**: Optimized dark-themed aesthetic (`#0a0a0a`) with cyan accents.
- **Performance**: High Lighthouse score, optimized font loading (`Outfit` + `Work Sans`), and proper SEO metadata.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run dev server:**
   ```bash
   npm run dev
   ```

## Tech Stack

- **Framework:** Next.js 15
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion
- **Fonts:** Google Fonts (Outfit, Work Sans)
- **Deployment:** Vercel (Recommended)
